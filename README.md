✈️ Uçak Bileti Rezervasyon Sistemi
Bu proje, Java dili kullanılarak geliştirilmiş, nesne yönelimli programlama (OOP) mantığına dayanan bir uçak bileti yönetim simülasyonudur. Uygulama, kullanıcıların bilet satın alma süreçlerini ve sadakat programı indirimlerini yönetir.

🌟 Öne Çıkan Özellikler

Sadakat Programı (5. Bilet İndirimi): Sistem, kullanıcının satın alma geçmişini takip eder. Aynı kullanıcı 5. biletini satın aldığında otomatik olarak özel indirim uygulanır.

Hata Yönetimi: Geçersiz veri girişlerine (negatif mesafe, hatalı karakter vb.) karşı kullanıcıyı uyaran kontrol mekanizmaları içerir.

Dinamik Fiyatlandırma: Mesafe ve kullanıcı geçmişine göre anlık tutar hesaplaması yapılır.
Dosya Tabanlı Veri Yönetimi: Tüm bilet bilgileri arka planda dosyalara kaydedilir ve işlemler bu dosyalar üzerinden yürütülür.

Bilet İade Sistemi: Satın alınan bilet iptal edilebilir. İade işlemi gerçekleştiğinde ilgili bilet dosyadan silinir ve ayrılan koltuk tekrar boşalır.

Bilet Çıktısı: İşlem başarıyla tamamlandığında bilet bilgileri en son ekrana yansıtılır.
🛠 Kullanılan Teknolojiler
Dil: Java

Yapı: Konsol Uygulaması (CLI) / [Eğer GUI varsa: Swing/JavaFX]
